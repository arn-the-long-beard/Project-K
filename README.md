### How to use ###

**Download**

    git clone https://github.com/arn-the-long-beard/Project-K.git

**Go to the directory**
   
    cd Project-Kk
    
**Install Npm Packages**
    
    npm install 
    
**Start the dev server**
  
    npm start
    
**For running the production server**

    npm run start_prod  
  
**In case you need to build** 

    npm run build
